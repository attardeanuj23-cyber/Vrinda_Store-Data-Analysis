# Vrinda_Store-Data-Analysis
The project Vrinda Store Data Analysis (Annual Report 2022) is a comprehensive data analytics study focused on evaluating the sales performance and customer behaviour of an e-commerce retail store over the course of a year.

The primary objective of the project is to analyze a dataset of over 31,000 transactions to provide actionable insights that can help the business improve its sales strategies, target the right audience, and optimize its distribution channels.

## Dataset used
- <a href="https://github.com/ritikbh193/Data-Analysis-Dashboard/blob/main/Vrinda%20Data%20Analysis2.xlsx">Dataset<a/>

Questions (KPIs)
- Compare the sales and orders using single chart.
- Which month got the highest sales and orders?
- Who purchased more - Men or Women?
- What are different order status in 2022?
- List top 10 states contributingto the sales?
- Relation between age and gender based on number of orders.
- Which Channel is contributing maximum to the sales?
- Highest selling category?
- Percentage of Total Orders delivered
- Dashboard Interaction <a href="https://github.com/attardeanuj23-cyber/Vrinda_Store-Data-Analysis/blob/main/Screenshot%202026-01-07%20135927.png">View Dashboard<a/>

Process
- Verify data for any missing values and anomalies, and sort out the same.
- Made sure data is consistent and clean with respect to data type, data format and values used.
- Created pivot tables according to the questions asked.
- Merge all pivot tables into one dashboard and apply slicer to make dynamic.
